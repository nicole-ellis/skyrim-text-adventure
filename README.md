# Skyrim Text Adventure

![ASCII Art](ascii-art)

A small terminal-based text adventure inspired by Skyrim, built in C++ as part of Codecademy’s “Build a Text Adventure” project.

The game allows players to choose between different classes and follow branching story paths through interactive dialogue and player choices.

## Features

- Warrior, Mage, and Thief class selection
- Branching dialogue paths
- Input validation loops
- ASCII art title screen
- UTF-8 console support for Unicode rendering
- Terminal-based gameplay

## Learning Outcomes

This project was created to practice:
- C++ conditionals
- Boolean logic
- Loops
- User input handling

## Running the Project

Compile the program:

```bash
g++ text-adventure.cpp -o text-adventure.exe
```

Run the executable:

```bash
./text-adventure.exe
```
